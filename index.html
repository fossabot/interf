INTERF

A JAVASCRIPT LIBRARY FOR DETERMINE WHETHER A OBJECT IS AN INSTANTIATED OF A CLASS THAT IMPLEMENTS AN INTERFACE

Download

  • Core build (~2kB gzipped)
  • Full build (~4kB gzipped)
  • CDN copies

What is INTERF?

Interf is fast and small javascript library.
It makes possible to determine whether a object is an instantiated of a class that implements an interface.
And it is possible to copy class and class.prototype properties by mix().in() too.

Installation

Using npm:

$ npm install --save interf

In Node.js:

const interf = require('interf');

In a browser:

<script src="interf.js"></script>

Support

Tested in:

  • Chrome 58 - 59
  • Firefox 54
  • IE 9 - 11
  • Safari 5.1.7
  • Opera 45
  • Node.js 6
  • PhantomJS 2.1.14

Basic usage

// create interface
const Quackable = interf.create('Quackable');
// bind interface
const Duck = interf.implement(Quackable).in(class Duck {});
// create instance
const donald = new Duck();
// test instance
Quackable.isInterfaceOf(donald); // true
